Travis Built Documentation documentation
========================================


Installation
------------

Travis Built Documentation can be installed in a handful of ways.

1. Official release *(Recommended)*

   .. code-block:: console

       $ pip install Travis Built Documentation

   If you don't have `pip`_ installed, this `Python installation guide`_
   can guide you through the process.


2. Development sources

   .. code-block:: console

       $ pip install git+https://github.com/has2k1/travis_doc.git

   Or

   .. code-block:: console

       $ git clone https://github.com/has2k1/travis_doc.git
       $ cd travis_doc
       $ python setup.py install

   Or

   .. code-block:: console

       $ curl -OL https://github.com/has2k1/travis_doc/archive/master.zip
       $ unzip master
       $ cd travis_doc-master
       $ python setup.py install
